*     Explication de la méthode de généralisation cartographique       *
*                          EDI_CONSTRUCTION_P                          *
************************************************************************

# Résumé :
Pas d'algorithmes de généralisation pour cette couche. Pour cette couche,  l'affichage se fait jusqu'à l'échelle 1/25000. Pour les échelles inférieures ( 1/50000 à 1/500000), La couche EDI_CONSTRUCTION_P ne figure pas sur la carte.


# Remarques et pistes d'amélioration : 

-Possibilité d'inscrire dans la base de donnée un champ importance permettant de faire une sélection sur des échelles plus faible que le 1/25000.

- Pour les antennes, possibilité d'ajouter un attribut HAUTEUR pour ainsi filtrer les antennes selon leur hauteur et ne garder que les plus grandes , pour ainsi empêcher leur superposition lors du changement d'échelle.
